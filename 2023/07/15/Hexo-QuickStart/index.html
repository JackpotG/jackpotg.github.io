GitHub Pages QuickStart

Quick Start

Create a static website

Create Repository

  1. Create a repository whit <username>.github.io
  2. Settings -> Pages
  3. ‘Build and deployment’ Section -> Deploy from a branch
  4. Save

Deploy to GitHub

  1. Install hexo-deployer-git

  2. Modify _config.yml

  3. hexo generate hexo clean hexo deploy

    1.  $ hexo generate --deploy
 $ hexo deploy --generate
 $ hexo g -d
 $ hexo d -g
