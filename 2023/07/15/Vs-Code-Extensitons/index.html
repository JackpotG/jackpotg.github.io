Markdown Preview Enhanced

Refer to the official documentation - 中文简体

Vs Code 安装

Extension -> markdown-preview-enhanced -> install

这个插件可以实时的显示Markdown的结果，个人感觉对于经常使用MD来写文档的人来说挺实用的。

关于编写

  • 基本元素 -> MD基本语法 + 扩展语法 + 自定义样式
  • 数学公式
  • 图像
  • TOC -> 通过 cmd-shift-p 然后选择 Markdown Preview Enhanced: Create Toc 命令来创建 TOC，若要排除某些标题 {ignore=true}
  • 导入文件
  • Code Chunk 渲染Js
  • 幻灯片
  • Pandoc 文档导出

文档导出

右键预览窗口 -> 选择到处格式
