Development Log

2023 - 07 - 15

近两天忽然想起来没有搭建出来一直想要的个人站点这个事情，然后在伟大的互联网上面寻求解决方案。目前定下来的solution是通过hexo + github的方式搭建一个静态站点，通过不断部署触发github pages deployment action来实现站点资源的更新。

对于站点初步设想是：此个人站点作为自用的互联网入口站点，通过内置各种超链接跳转至自己需要的一些资源网站和常用站点，比如个人博客。

Quick Start

Create a static website

Create Repository

  1. Create a repository whit <username>.github.io
  2. Settings -> Pages
  3. ‘Build and deployment’ Section -> Deploy from a branch
  4. Save

Deploy to GitHub

  1. Install hexo-deployer-git

  2. Modify _config.yml

  3. hexo generate hexo clean hexo deploy

    1.  $ hexo generate --deploy
 $ hexo deploy --generate
 $ hexo g -d
 $ hexo d -g
